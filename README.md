# Retail-AI-Assistant-Assignment
Retail AI Assistant Assignment for the role of AI Developor

**Steps to run :**
1. Clone the repo to local
2. Change directories to the repo location
3. For WINDOWS : set OPENAI_API_KEY=value
4. For Linux : Export OPENAI_API_KEY=value
5. Run the agent using : python agent.py

**Files included : **
1. agent.py
2. tools.py
3. products.csv
4. oorders.csv
5. policy.txt
6. Architecture.docx
7. Retail_AI_Assignment.mp4
